# Cornerfolio Frontend

## Run the development server:

```sh
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Build the application:

```sh
npm run build
# or
yarn build
# or
pnpm build
# or
bun build
```

## Run the production application:

__NOTE: You must build first in order to run this command__

```sh
npm run start
# or
yarn start
# or
pnpm start
# or
bun start
```

## Check Link Errors:

```sh
npm run lint
# or
yarn lint
# or
pnpm lint
# or
bun lint
```
